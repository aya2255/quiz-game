# Quiz Game

A simple and interactive Quiz Game built using **HTML**, **CSS**, and **JavaScript**.

## Features

- Start screen
- Multiple-choice questions
- Score calculation
- Progress bar that updates after each correct answer
- Final result screen displaying the user's score
- Clean and responsive UI

##  Technologies Used

- HTML5
- CSS3
- JavaScript
